# Memory Bank Root

Este directorio se usa como `MEMORY_BANK_ROOT` para `@allpepper/memory-bank-mcp`.

El servidor MCP puede crear y mantener aqui la estructura del banco de memoria por proyecto.

- Nombre sugerido del proyecto: `Proyecto ERP MODULAR Bocam`
- Estado actual: MCP instalado localmente y configurado en el workspace
- Directorio del proyecto: `.memory-bank/Proyecto ERP MODULAR Bocam`
